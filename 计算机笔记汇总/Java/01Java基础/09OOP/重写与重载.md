**方法的重写（override）两同两小一大原则**：

- 方法名相同，参数类型相同
- 子类返回类型小于等于父类方法返回类型，
- 子类抛出异常小于等于父类方法抛出异常，
- **子类访问权限大于等于父类方法访问权限**。
