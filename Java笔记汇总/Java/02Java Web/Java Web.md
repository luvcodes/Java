# JavaWeb技术栈

- B/S架构: Browser/Server，浏览器/服务器架构
- 静态资源: HTML, CSS, JS, 图片等。负责页面展现
- 动态资源: Servlet, JSP等。负责逻辑处理

# JavaWeb的概念

## 什么是Java Web？

所有通过Java语言编写可以通过浏览器访问的程序的总称

它是基于请求和相应来开发的

## 什么是请求？

客户端发送给服务器的数据叫做请求

## 什么是响应？

服务器给客户端回传数据叫响应

## Web资源的分类

- 静态资源
- 动态资源
